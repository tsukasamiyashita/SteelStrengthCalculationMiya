# SteelStrengthCalculationMiya

鋼材・ボルト・ワイヤーロープの断面性能および強度を計算するブラウザベースのアプリケーションです。
実行すると自動的にブラウザが立ち上がり、すぐに計算を開始できます。

## 特徴
- **3つの計算モード**を切り替えて使用でき、多様な強度計算に1つのアプリで対応します。
- 一般的なJIS規格等の寸法データや物性値をリスト（データベース）化して内蔵しています。

## 各モードの機能
### 1. 鋼材の強度計算
- 「H鋼」「アングル」「パイプ」など全13種の形状に対応。
- **規格寸法（例：100 x 50 x 5 x 7）を選ぶだけ**で断面性能（断面積、断面二次モーメントなど）を算出。
- **断面の向き（強軸・弱軸）を選択可能**で、横倒しで使用した場合の強度計算やたわみ量計算に対応。

### 2. ボルトの強度計算
- M6〜M36までのメートル並目ねじに対応。
- 一般的な強度区分（4.6、4.8、8.8、10.9、12.9）を選ぶだけで降伏点・引張強さを自動取得。
- 安全率に基づく許容引張荷重、および許容せん断荷重を算出します。

### 3. ワイヤーロープの強度計算
- 代表的なロープ（6×24 A種、6×37 A種など）の破断荷重データを内蔵。
- 吊り本数と吊り角度（0〜120度）を指定することで、玉掛け作業時の「システム全体の許容荷重」を計算します。
