# SteelStrengthCalculationMiya

鋼材の断面性能および強度を計算するブラウザベースのアプリケーションです。
実行すると自動的にブラウザが立ち上がり、すぐに計算を開始できます。

## 対応している鋼材形状（全13種）
- H鋼
- I形鋼 (Iビーム)
- チャンネル（溝形鋼）
- リップみぞ形鋼（C型鋼）
- T形鋼
- 等辺アングル（山形鋼）
- 不等辺アングル
- フラットバー（平鋼）
- 四角棒
- 角パイプ（角形鋼管）
- 丸パイプ（円形鋼管）
- 丸棒（丸鋼）
- 六角棒
