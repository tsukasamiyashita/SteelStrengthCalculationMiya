# SteelStrengthCalculationMiya

鋼材の断面性能および強度を計算するブラウザベースのアプリケーションです。
実行すると自動的にブラウザが立ち上がり、すぐに計算を開始できます。

## 特徴
- 一般的なJIS規格等の鋼材寸法をリスト（データベース）化して内蔵しています。
- 「H鋼」「アングル」「パイプ」などの形状を選び、**規格寸法（例：100 x 50 x 5 x 7）をドロップダウンリストから選択するだけ**で計算できます。
- **断面の向き（強軸・弱軸）を選択可能**になり、横倒しで使用した場合の強度計算にも対応しました。

## 対応している鋼材形状（全13種）
- H鋼
- I形鋼 (Iビーム)
- チャンネル（溝形鋼）
- リップみぞ形鋼（C型鋼）
- T形鋼
- 等辺アングル（山形鋼）
- 不等辺アングル
- フラットバー（平鋼）
- 四角棒
- 角パイプ（角形鋼管）
- 丸パイプ（円形鋼管）
- 丸棒（丸鋼）
- 六角棒
